Upskilling / Reskilling API corrigida

- .NET 9
- Oracle + EF Core 9.0.0 estável
- Paginação + HATEOAS
- CRUD completo para TB_CADASTRO, TB_MEUPROGRESSO, TB_OBJETIVOS
- JWT + Swagger
- Health Check + Logging + Tracing + Cache em memória

Passos básicos:
1. Abrir UpskillingApi_fixed no Visual Studio 2022 (abrir o .csproj).
2. Restaurar pacotes NuGet.
3. Ajustar se necessário as versões dos pacotes para as existentes no seu ambiente.
4. Rodar a aplicação (F5) e usar o Swagger em /swagger.
